# Game_24points
24 點 撲克牌挑戰  
 
每次有4張撲克牌，利用加減乘除使結果為24。  


20260518  
本次更新重點說明：    
1. 難易度分級邏輯：  
- 在「挑戰模式」中，點擊開始時會進入一個 while 迴圈進行預計算。  
- 系統會自動過濾解法組數。例如選擇「SSS」，系統會持續抽牌直到找到一組解法在 1~3 組之間的題目。  
- 是否包含無解：新增了 allow_no_sol 開關（預設關閉）。關閉時，系統會確保抽出來的牌一定有解且符合難度。  
2. 全面展示解答：  
- 無論是挑戰模式還是計算機，現在都會使用 st.code 塊列出所有不重複的數學組合。    
- 使用了遞迴結合 set 資料結構，過濾掉完全相同的運算路徑。    
3. 計算機優化：  
- 移除氣球：計算機模式現在只會安靜地顯示結果，不會觸發 st.balloons()。    
- 多牌支援：解牌計算機依然支援彈性的張數輸入。    
4. 介面小調整：  
- 將遊戲設定移動到 st.expander 中，讓主畫面更專注於撲克牌顯示。  
- 增加了難度選單與對應的組數說明。
